1. HTML (Structure)

Insights: The structure focuses on simplicity and keeping the user experience intuitive. A <main> section was used to logically group the story text, choices, and potential dynamic elements like the addendum for clarity and accessibility. The <footer> includes the lastModified functionality to offer transparency about updates.

Challenges: Maintaining semantic correctness while integrating a dynamically updating section like the addendum. Ensuring the new section fits seamlessly into the overall layout without disrupting existing elements.

Creative Process: Positioned the addendum inside the <main> tag to keep it aligned with the interactive elements. Used semantic tags like <header>, <main>, and <footer> to enhance accessibility and organization.

2. CSS (Styling)

Insights: The design prioritizes readability and aesthetic coherence, using colors like soft green (#effaf3) to align with the forest adventure theme. Rounded borders and padding create a modern, approachable interface, while typography ('Verdana', sans-serif) ensures readability.

Challenges: Balancing aesthetic appeal with functionality and avoiding layout shifts caused by dynamic content additions like the addendum.

Creative Process: Used margin-top and padding to separate the main content from the addendum. Defined a color palette that complements the theme without overwhelming the user. Ensured responsiveness by using flexible units like rem and %.
